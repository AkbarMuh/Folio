# Folio — Akbar Muhammad Prakoso

Personal portfolio of **Akbar Muhammad Prakoso**, Applied AI Engineer.
A single-page, **Persona 5-styled** site built with plain HTML, CSS, and vanilla JavaScript — no build step, no framework.

🔗 **Live:** https://akbarmuh.github.io/Folio/

## Features
- Fully self-contained `index.html` (inline CSS + JS)
- Persona 5 UI: all-out-attack section transitions, splash intro, synthesized Web Audio SFX (no audio files), animated hero
- Responsive (desktop → mobile) with reduced-motion support
- Sections: Home · About · Records (education & experience) · Heists (projects) · Arsenal (skills) · Contact

## Structure
```
index.html   → the site (self-contained)
404.html     → custom Persona 5 404 page
img/         → profile & about photos
PROFILE.md   → public-safe professional profile
```

## Deploy
Hosted on **GitHub Pages** from the `main` branch. Any push to `main` redeploys automatically.
